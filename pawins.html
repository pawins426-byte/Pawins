Questions Fréquentes
Comment déposer des USDT? +
Allez dans "Recharger", choisissez votre réseau (BEP20 ou TRC20), copiez l'adresse, envoyez vos USDT et entrez le hash de transaction. Activation sous 30 minutes.
Quand puis-je retirer? +
Les retraits s'effectuent chaque Samedi uniquement. Minimum $10 USDT. Vos profits s'accumulent quotidiennement jusqu'au samedi.
Comment fonctionne le parrainage? +
Partagez votre lien unique. Quand un ami achète un plan, vous recevez automatiquement 3 à 4% du montant de son plan dans votre solde.
Quel réseau USDT utiliser? +
Vous pouvez utiliser BEP20 (BNB Chain) ou TRC20 (TRON) pour les dépôts. Pour les retraits, nous utilisons uniquement le réseau BEP20.
Puis-je avoir plusieurs plans? +
Un plan actif par compte à la fois. Une fois le plan terminé, vous pouvez en choisir un nouveau ou le même.
Mon investissement est-il sécurisé? +
Oui. Toutes les transactions sont vérifiées via la blockchain. Votre compte est protégé par authentification sécurisée.
